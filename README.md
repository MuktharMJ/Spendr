#  Spendr

Spendr is a modern full-stack expense management application built with **React, Vite, Bun, Prisma, and Tailwind CSS**. It helps users efficiently manage daily expenses, organize spending records, and provides a scalable architecture for future financial management features.

---

##  Features

-  Expense tracking and management
-  Fast React + Vite frontend
-  Responsive UI with Tailwind CSS
-  Bun-powered backend
-  Prisma ORM database integration
-  Modular and reusable component architecture
-  TypeScript support
-  Production-ready project structure

---

### Frontend

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/MuktharMJ/Spendr.git
cd Spendr
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

### Run the complete development environment

```bash
npm run dev:full
```
